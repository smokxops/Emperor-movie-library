AN OOP Movie Library App

A simple JavaScript movie library using Object-Oriented Programming (OOP).
Users can search, add, rate, and delete movies using the OMDB API.

Features

. Search movies via OMDB API
. Add movies to your library
. Rate movies (1-10)
. Delete movies
. Data saved using LocalStorage

Built With

. JavaScript(ES6+)
. OOP (Classes, Inheritance)
. Async/Await
. Fetch API
. CRUD structure
. LocalStorage
. DOM manipulation
